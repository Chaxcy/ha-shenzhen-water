# 深圳水务 HA 插件

一个用于 Home Assistant 的深圳水务自定义集成，可通过深圳水务账号绑定手机号登录，自动获取户号、水费账单、用水量、水表读数、缴费状态等信息。

## 功能

* 支持 Home Assistant UI 配置，无需手动写 YAML
* 支持短信验证码登录
* 自动获取已绑定的深圳水务户号
* 自动生成并刷新 Ctoken
* 支持自定义数据同步间隔，单位为分钟
* 支持显示本期账单和上期账单数据
* 支持中文界面翻译

## 支持的实体

当前会创建以下传感器：

* 账单月份
* 缴费状态
* 到期日
* 本期用水量
* 当前水表读数
* 本期总费用
* 应缴金额
* 自来水费
* 污水处理费
* 垃圾费
* 上期账单月份
* 上期用水量
* 上期总费用

## 安装方式

通过 HACS 自定义仓库安装：

1. 打开 Home Assistant
2. 进入 HACS
3. 点击右上角菜单
4. 选择“自定义仓库”
5. 添加仓库地址：

```text
https://github.com/Chaxcy/ha-shenzhen-water
```

6. 类型选择：

```text
Integration
```

7. 安装后重启 Home Assistant

## 配置方法

重启 Home Assistant 后：

1. 进入“设置”
2. 进入“设备与服务”
3. 点击“添加集成”
4. 搜索“深圳水务”
5. 输入深圳水务绑定手机号
6. 设置同步间隔，单位为分钟
7. 获取短信验证码
8. 输入验证码完成添加

添加成功后，插件会自动获取绑定户号和最新账单信息。

## 同步间隔

同步间隔单位为分钟。

默认值：

```text
360 分钟
```

也就是 6 小时。

建议设置：

```text
360 - 720 分钟
```

水费账单不是实时数据，通常不需要高频刷新。过于频繁的请求可能增加接口限制或风控风险。

添加完成后，也可以进入集成的“配置”页面修改同步间隔。

## Ctoken 与登录状态

深圳水务接口中，账单相关接口需要 Ctoken。

本插件会自动：

1. 登录获取 Utoken
2. 获取绑定户号
3. 根据户号生成 Ctoken
4. 请求账单数据
5. 当 Ctoken 失效时自动重新生成并重试

如果 Utoken 登录状态失效，插件无法自动短信登录，需要删除集成后重新添加，并重新获取短信验证码。

## 常见问题

### 1. 添加集成时报验证码错误

请确认：

* 手机号为深圳水务账号绑定手机号
* 验证码未过期
* 没有多次重复请求验证码
* 手机号可以正常登录深圳水务网页端

### 2. 实体显示 unavailable

请查看 Home Assistant 日志。

如果日志中出现类似：

```text
深圳水务登录已失效
```

说明登录 token 已失效，需要删除该集成后重新添加。

### 3. Ctoken 参数缺失或不合法

插件会自动尝试重新生成 Ctoken 并重试一次。

如果仍然失败，请查看日志中的完整接口返回信息。
